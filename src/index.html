OBJECTIVE OF THE GAME

The objective of the game is to clear a minefield without detonating any!

RULES

  • The boxes have a number that indicates the number of mines in the surrounding cells

  • If you discover a square with a mine, the game is lost.

  • If a box with a 0 is found, it indicates that none of the neighboring cells has a mine and will be discovered automatically.

  • If you do not customize the game the board will be 15x15 and 38 bombs will be assigned.
